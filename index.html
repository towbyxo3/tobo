Command Description
Info
!avatar Displays the current and past profile pictures of members.
!serverinfo Returns information about the current server.
!userinfo Returns information about the user.
!names Returns the previous usernames/nicknames of the user.
!banner Returns the banner of the user.
!collectavatars Stores avatars of members in the current guild.
!about Returns stats and information about the bot.
!ping Returns the ping of the bot.
!invite Returns the invite link for the current server.
!myservers Prints invite links to the bot's servers in the console.
!github Returns the link to this GitHub repository.
!covid Returns COVID statistics of a country.
!country Returns general information about a country.
Leaderboard
!crowns Displays members with the most crowns.
!age Displays the youngest or oldest members based on a criteria.
!chat Displays members with the most messages in selected timeframe.
!mentions Displays mention leaderboard and stats of the server and member.
!serverpeak Displays timeframes where most messages were sent in the server.
!userpeak Displays timeframes where a member sent the most messages in the server.
Mod
!kick Kicks a member from the server.
!nickname Changes the nickname of a member.
!ban Bans a member from the server.
!massban Bans several members from the server.
!unban Unbans a user from the server.
!prune embeds Removes a certain number of recently sent embeds in the chat.
!prune files Removes a certain number of recently sent files in the chat.
!prune mentions Removes a certain number of recently sent mentions in the chat.
!prune images Removes a certain number of recently sent images in the chat.
!prune all Removes every recently sent message.
!prune user Removes a certain number of recently sent messages by a member.
!prune contains Removes a certain number of recently sent messages that contain a phrase.
!prune bots Removes a certain number of recently sent messages by bots.
!prune users Removes a certain number of recently sent messages by users.
!prune reactions Removes a certain number of recently added reactions.
Fun
!network Returns the most frequent chat encounters of a member graphically.
!wordcloud Returns a wordcloud that shows the most frequently used words by a member graphically.
!wordcloudserver Returns a wordcloud that shows the most frequently used words in the server graphically.
!said Returns members who most frequently used the word.
!rewind Returns an embed with clickable rewind pictures and stats of a year.
!rewindserver Returns an embed with clickable rewind pictures and stats of a year.
!quote Returns a random message (by a member) that contains a certain phrase or word.
!snipe Returns the last deleted message (by a member).
!snip Returns the last edited message (by a member).
Voicechat
!massmove Moves members in the current channel to a certain channel.
!massmoveall Moves all members who are in a voice channel to a certain channel.
Admin
!owner Returns the owner.
!load Loads an extension.
!unload Unloads an extension.
!reload Reloads an extension.
!reloadall Reloads all extensions.
!reloadutils Reloads utils.
!change playing Change the status of the bot.
!change username Change the username of the bot.
!change nickname Change the nickname of the bot.
!change avatar Change the avatar of the bot.
